정수 2개를 입력받아
합을 출력하는 프로그램을 작성해보자.



- 입력

  ```
  2개의 정수가 공백으로 구분되어 입력된다.
  
  
  ```
  
- 출력

  ```
  두 정수의 합을 출력한다.
  ```

- 입력 예시

  ``` python
  123 -123
  ```

  

- 출력 예시

  ```python
  0
  ```

  

---

- 답변 

  ``` python
  #키보드 입력값을 통해 문자열을 입력받는다
  a, b = input().split()
  s = int(a) + int(b) #2개의 정수를 더하여야하기 때문에 정수로 형변환후 더해준다
  print(s)
  ```

  

- 모범 답변

  ``` python
  a, b = input().split()
  a=int(a)
  b=int(b)
  c=a+b
  print(c)
  
  
  ```
