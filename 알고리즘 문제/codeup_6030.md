영문자 1개를 입력받아 10진수 유니코드(Unicode) 값으로 출력해보자.

예시
n = ord(input())
print(n)

- 입력

  ```
  유니코드 영문자 1개가 입력된다.
  
  
  ```
  
- 출력

  ```
  유니코드 값을 10진수로 출력한다.
  ```

- 입력 예시

  ``` python
  A
  ```
  
  
  
- 출력 예시

  ```python
  65
  ```

  

---

- 답변 

  ``` python
  n = ord(input())
  print(n)
  ```
  
  
  
- 모범 답변

  ``` python
  n = ord(input())
  print(n)
  
  ```
