줄을 바꿔 문자(character) 2개를 입력받고, 순서를 바꿔 한 줄씩 출력해보자.

- 입력

  ```
  2개의 문자가 줄을 바꿔 입력된다.
  
  ```

- 출력

  ```
  순서를 바꿔 한 줄에 한 문자씩 출력한다.
  
  ```

- 입력 예시

  ``` python
  a
  b
  ```

  

- 출력 예시

  ```python
  b
  a
  ```

  

---

- 답변 

  ``` python
  a = input() #키보드 입력값을 입력해준다
  b = input()
  print(b) # 출력 예시에 b가 먼저 출력되었으니 순서를 바꿔 b부터 출력을 해준다
  print(a) 
  
  ```

  

- 모범 답변

  ``` python
  a=input() # 정답은 이렇지만 정수형을 출력하라 하였으니 
  		  # 정수형으로 변환해주는게 맞다고 생각한다
  b=input()
  print(b)
  print(a)
  ```
