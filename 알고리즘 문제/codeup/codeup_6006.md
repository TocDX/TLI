이번에는 특수문자 출력에 도전하자!!

다음 문장을 출력하시오.

"!@#$%^&*()'
(단, 큰따옴표와 작은따옴표도 함께 출력한다.)

- 입력

  ```
  입력 없음
  ```

- 출력

  ```
  "!@#$%^&*()'
  를 출력한다.
  ```

- 입력 예시

- 출력 예시

  ```python
  "!@#$%^&*()'
  ```

  

---

- 답변 

  ``` python
  print("\"!@#$%^&*()'") # -> 문자열 출력 할 때 따옴표 안에 따옴표를 넣고 싶은 경우에 
  						#	\ 백슬러시 이스케이프 문자를 활용한다
  ```

  

- 모범 답변

  ``` python
  print("\"!@#$%^&*()'")
  또는
  print('"!@#$%^&*()\'')
  
  ```
