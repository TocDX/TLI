10진 정수 1개를 입력받아
유니코드 문자로 출력해보자.


예시
c = int(input())
print(chr(c)) #c에 저장되어 있는 정수 값을 유니코드 문자(chracter)로 바꿔 출력한다. 

- 입력

  ```
  10진 정수 1개(32 ~ 126 범위)가 입력된다.
  
  
  ```
  
- 출력

  ```
  유니코드 문자로 출력한다.
  ```

- 입력 예시

  ``` python
  65
  ```
  
  
  
- 출력 예시

  ```python
  A
  ```

  

---

- 답변 

  ``` python
  c = int(input())
  print(chr(c))
  ```
  
  
  
- 모범 답변

  ``` python
  c = int(input())
  print(chr(c))
  ```
