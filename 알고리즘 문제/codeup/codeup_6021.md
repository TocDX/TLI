알파벳과 숫자로 이루어진 단어 1개가 입력된다.
입력받은 단어의 각 문자를 한 줄에 한 문자씩 분리해 출력한다.



- 입력

  ```
  5개의 문자로 이루어진 단어 1개가 입력된다.
  
  
  ```

- 출력

  ```
  각 문자를 한 줄에 한 문자씩 줄을 바꿔 출력한다.
  
  ```

- 입력 예시

  ``` python
  Hello
  ```

  

- 출력 예시

  ```python
  H
  e
  l
  l
  o
  ```

  

---

- 답변 

  ``` python
  a= input()
  #키보드 입력값을 통해 문자열을 입력받는다
  print(a[0])
  print(a[1])
  print(a[2])
  print(a[3])
  print(a[4])
  # 각 문자를 한 줄에 한 문자씩 출력해준다
  ```

  

- 모범 답변

  ``` python
  s = input()
  print(s[0])
  print(s[1])
  print(s[2])
  print(s[3])
  print(s[4])
  
  ```
