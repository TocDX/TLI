정수(integer), 실수, 문자(character), 문자열(string) 등 1개만 입력받아 한 줄로 3번 출력해보자.



- 입력

  ```
  1개의 데이터가 입력된다.
  
  ```

- 출력

  ```
  공백을 두고 3번 출력한다.
  
  
  ```

- 입력 예시

  ``` python
  computer science
  ```

  

- 출력 예시

  ```python
  computer science computer science computer science
  ```

  

---

- 답변 

  ``` python
  a = input() #키보드 입력값을 통해 문자열을 입력받는다
  print(a, a, a) # 출력 예시처럼 ,띄어쓰기로 세 번 반복한다
  ```

  

- 모범 답변

  ``` python
  a=input()
  print(a, a, a)
  
  ```
