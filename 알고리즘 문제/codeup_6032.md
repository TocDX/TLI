입력된 정수의 부호를 바꿔 출력해보자.


예시 
... 
print(-n) 

- 입력

  ```
  정수 1개가 입력된다.
  
  ```
  
- 출력

  ```
  부호를 바꿔 출력한다.
  ```

- 입력 예시

  ``` python
  -1
  ```
  
  
  
- 출력 예시

  ```python
  1
  ```

  

---

- 답변 

  ``` python
  i = input()
  
  n = int(i)
  
  print(-n)
  ```
  
  

- 모범 답변

  ``` python
  a=int(input())
  print(-a)
  
  ```
