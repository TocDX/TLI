이번에는 줄을 바꿔 출력하는 출력문을 연습해보자.
다음과 같이 줄을 바꿔 출력해야 한다.

Hello
World
(두 줄에 걸쳐 줄을 바꿔 출력)

- 입력

  ```
  입력 없음
  ```

- 출력

  ```
  Hello
  World
  를 출력한다
  ```

- 입력 예시

- 출력 예시

  ```python
  Hello
  World
  ```

  

---

- 답변 

  ``` python
  print("Hello")
  print("World") -> 각각의 프린트문에 Hello World 입력해준다
  ```

  

- 모범 답변

  ``` python
  print("Hello")
  print("World")
  ```
