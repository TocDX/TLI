시:분:초 형식으로 시간이 입력될 때 분만 출력해보자.

어떻게 분만 출력해야 할지 주의 깊게 생각해야한다.



- 입력

  ```
  시 분 초가
  시:분:초 형식으로 입력된다.
  
  
  ```

- 출력

  ```
  분만 출력한다.
  ```

- 입력 예시

  ``` python
  17:23:57
  ```

  

- 출력 예시

  ```python
  23
  ```

  

---

- 답변 

  ``` python
  #키보드 입력값을 통해 문자열을 입력받는다
  h, m, s = input().split(':')
  # 각 시, 분, 초에 :을 쪼개어 값을 주고
  print(m)
  #분만 필요하다 하였으니 m만 출력해준다
  ```

  

- 모범 답변

  ``` python
  h, m, s = input().split(':')
  print(m)
  
  
  ```
