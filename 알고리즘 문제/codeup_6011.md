숫자(0~9)와 소수점(.)을 사용해 표현한 수를 실수(real number)라고 한다.

변수에 실수값을 저장한 후
변수에 저장되어 있는 값을 그대로 출력해보자.

- 입력

  ```
  실수 1개가 입력된다.
  ```

- 출력

  ```
  입력된 수를 실수로 변환하여 출력한다.
  
  ```

- 입력 예시

  ``` python
  1.414213
  ```

  

- 출력 예시

  ```python
  1.414213
  ```

  

---

- 답변 

  ``` python
  a = input() #키보드 입력값을 입력해준다
  a = float(a) # 출력할 때 실수가 나와야하니 문자열을 실수형으로 변환해준다
  print(a) 
  
  ```

  

- 모범 답변

  ``` python
  a = input()
  a = float(a)
  print(a)
  ```
